# BirdFlock
A graphics program simulating Brownian Motion;

### implementations
random motion.

### Dependencies:
- glfw (window)
- Vulkan (graphics api)
- glm (math library)
- stb_image.h (for importing images: https://github.com/nothings/stb/blob/master/stb_image.h)
